01. 슬라이드 이펙트 : 트랜지션 효과

javascript와 jquery를 사용해 슬라이더를 구현했습니다.

javascript :
setInterval을 사용하여 3초마다 슬라이드가 전환되도록 설정되어 있습니다.
currentIndex 변수를 사용하여 현재 이미지의 인덱스를 추적하고, nextIndex 변수를 사용하여 다음 이미지의 인덱스를 계산합니다.
각 이미지의 투명도를 조절하여 이미지가 나타나고 사라지는 효과를 구현합니다.

jquery :
setInterval을 이용하여 3초마다 슬라이드가 전환되도록 설정되어 있습니다.
.fadeOut() 및 .fadeIn() 메서드를 사용하여 이미지를 나타나고 사라지게 합니다.

02. 슬라이드 이펙트 : 좌로 움직이기

javascript와 jquery를 그리고 GSAP를 사용해 슬라이더를 구현했습니다.

slider__inner는 이미지가 좌우로 움직이는 영역으로, 각 이미지의 총 너비가 4000px입니다. slider__img는 이미지가 보이는 영역으로 overflow: hidden을 주어 해당 슬라이더의 이미지만 나타나게 구현했습니다.

javascript :
setInterval 함수를 활용하여 이미지를 일정 시간 간격으로 변경하고, 이때 translateX를 이용하여 이미지를 좌측으로 이동시킵니다.

jquery :
setInterval 함수를 이용하여 일정 시간 간격으로 이미지를 변경하며, animate 함수를 사용하여 좌측으로 이동시킵니다.

GSAP :
gsap.to 함수를 활용하여 이미지 이동 애니메이션을 적용하며, ease 옵션을 통해 이징 효과를 부여합니다.

03. 슬라이드 이펙트 : 좌로 움직이기(연속적으로)

javascript와 jquery를 그리고 GSAP를 사용해 슬라이더를 구현했습니다.

slider__inner는 이미지가 좌우로 움직이는 영역으로, 각 이미지의 총 너비가 4000px입니다. slider__img는 이미지가 보이는 영역으로 overflow: hidden을 주어 해당 슬라이더의 이미지만 나타나게 구현했습니다.
첫 번째 이미지를 복사하고, 이 복사본을 마지막에 붙여넣어 무한 슬라이더를 구현했습니다. 이는 슬라이드가 마지막 이미지까지 도달하면 첫 번째 이미지로 순환되어 움직이는 효과를 만듭니다.

javascript :
setInterval 함수를 사용하여 일정 시간 간격으로 이미지를 변경하며, translateX 속성을 조정하여 이동합니다.

jquery :
animate 함수를 사용하여 이미지를 좌측으로 이동시킵니다.

GSAP :
gsap.to 함수를 이용하여 x 속성을 조정하여 이미지를 이동시킵니다.

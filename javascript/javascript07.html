01. 서치 이펙트 : indexOf()/search()를 이용하여 검색하기

indexOf() 메서드는 JavaScript에서 문자열에서 특정 문자열 또는 문자의 첫 번째 발생 위치를 찾는 데 사용되는 메서드입니다.
만약 찾고자 하는 문자열이 존재하지 않는 경우에는 -1을 반환합니다.

search() 메서드는 JavaScript에서 문자열에서 정규 표현식과 매치되는 첫 번째 패턴의 인덱스를 반환하는 메서드입니다.
매치되는 패턴이 없으면 -1을 반환합니다.
search() 메서드는 정규 표현식을 사용하여 패턴을 지정하므로, 보다 유연한 문자열 검색이 가능합니다.

02. 서치 이펙트 : includes를 이용하여 검색하기

includes() 메서드는 JavaScript에서 사용되는 문자열 메서드 중 하나로, 문자열에 특정 문자열이 포함되어 있는지 여부를 확인합니다.
이 메서드는 불리언 값을 반환하며, 포함되어 있으면 true를 반환하고, 그렇지 않으면 false를 반환합니다.
대소문자를 구분하며, 부분 문자열이 일치하는 경우에도 true를 반환합니다.

03. 서치 이펙트 : charAt()를 이용하여 검색하기

charAt() 메서드는 JavaScript에서 문자열에서 특정 인덱스에 위치한 문자를 반환하는 메서드입니다. 이 메서드는 문자열의 각 문자에 대해 인덱스를 기반으로 해당하는 문자를 반환합니다.
문자열의 인덱스는 0부터 시작하며, 메서드의 매개변수로 전달된 인덱스에 해당하는 문자를 반환합니다.
유효한 인덱스를 전달하면 해당 인덱스에 위치한 문자를 반환하며, 범위를 벗어나는 인덱스를 전달하면 빈 문자열("")을 반환합니다.

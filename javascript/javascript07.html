01. 서치 이펙트 : indexOf()/search()를 이용하여 검색하기

indexOf() 메서드는 JavaScript에서 문자열에서 특정 문자열 또는 문자의 첫 번째 발생 위치를 찾는 데 사용되는 메서드입니다.
만약 찾고자 하는 문자열이 존재하지 않는 경우에는 -1을 반환합니다.

search() 메서드는 JavaScript에서 문자열에서 정규 표현식과 매치되는 첫 번째 패턴의 인덱스를 반환하는 메서드입니다.
매치되는 패턴이 없으면 -1을 반환합니다.
search() 메서드는 정규 표현식을 사용하여 패턴을 지정하므로, 보다 유연한 문자열 검색이 가능합니다.

02. 서치 이펙트 : includes를 이용하여 검색하기

includes() 메서드는 JavaScript에서 사용되는 문자열 메서드 중 하나로, 문자열에 특정 문자열이 포함되어 있는지 여부를 확인합니다.
이 메서드는 불리언 값을 반환하며, 포함되어 있으면 true를 반환하고, 그렇지 않으면 false를 반환합니다.
대소문자를 구분하며, 부분 문자열이 일치하는 경우에도 true를 반환합니다.

03. 서치 이펙트 : charAt()를 이용하여 검색하기

charAt() 메서드는 JavaScript에서 문자열에서 특정 인덱스에 위치한 문자를 반환하는 메서드입니다. 이 메서드는 문자열의 각 문자에 대해 인덱스를 기반으로 해당하는 문자를 반환합니다.
문자열의 인덱스는 0부터 시작하며, 메서드의 매개변수로 전달된 인덱스에 해당하는 문자를 반환합니다.
유효한 인덱스를 전달하면 해당 인덱스에 위치한 문자를 반환하며, 범위를 벗어나는 인덱스를 전달하면 빈 문자열("")을 반환합니다.

04. 서치 이펙트 : find()를 이용하여 검색하기

find() 메서드는 배열에서 주어진 콜백 함수의 조건을 만족하는 첫 번째 요소를 찾아 반환합니다.
이 메서드는 배열의 각 요소에 대해 주어진 함수를 호출하고, 함수가 true를 반환하는 첫번째 요소를 반환합니다.
만약 어떤 요소도 조건을 만족하지 않으면 undefined를 반환합니다. 화살표 함수를 사용할 수 있습니다.
callback: 배열의 각 요소에 대해 실행되는 함수로서, 세 가지 인자를 받습니다.
element: 현재 처리 중인 요소.
index: 현재 처리 중인 요소의 인덱스.
array: find()를 호출한 배열.

05. 서치 이펙트 : filter()를 이용하여 검색하기

filter() 메서드는 배열의 각 요소를 순회하면서 주어진 함수의 조건을 만족하는 요소들로 이루어진 새로운 배열을 생성합니다.
이 메서드는 기존 배열을 변경하지 않고 새로운 배열을 반환합니다. 화살표 함수를 사용할 수도 있습니다.
callback: 배열의 각 요소에 대해 실행되는 함수로서, 세 가지 인자를 받습니다.
element: 현재 처리 중인 요소.
index: 현재 처리 중인 요소의 인덱스.
array: filter()를 호출한 배열.

06. 서치 이펙트 : sort(), reverse()를 이용하여 검색하기

sort() 메서드는 배열의 요소를 정렬할 때 사용됩니다. 기본적으로는 문자열로 변환한 각 요소의 유니코드 코드 포인트 순서에 따라 정렬됩니다.
그러나 정렬 순서를 변경하려면 정렬 함수를 제공할 수 있습니다.
sort() 메서드에서 사용되는 정렬 함수는 선택적으로 제공할 수 있는 함수로서, 이 함수는 두 개의 인자를 받아서 비교하고 정렬 순서를 결정합니다.
함수는 일반적으로 다음과 같은 형태를 가집니다.
sort() 메서드를 사용할 때 주의할 점은 기본 동작은 문자열로 정렬되므로 숫자 배열의 경우 정렬 함수를 제공하여 숫자로 올바르게 정렬되도록 해야 합니다..
그리고 원본 배열을 직접 변경한다는 점을 주의해야합니다. 

{
  function compareFunction(a, b) {
    // 비교 로직을 구현하여 정렬 순서를 결정
    // a와 b를 비교하여 a가 b보다 작으면 음수, 크면 양수, 같으면 0을 반환
    // 예: 오름차순 정렬 => return a - b;
    // 예: 내림차순 정렬 => return b - a;
    }
}

reverse() 메서드는 배열의 요소 순서를 반대로 뒤집습니다. 즉, 배열의 첫 번째 요소는 마지막 요소가 되고, 마지막 요소는 첫 번째 요소가 됩니다.
이 메서드는 기존의 배열을 변경하며, 새로운 배열을 반환하지 않습니다.
